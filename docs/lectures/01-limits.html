1 Предел функции

1.1 Неформальная идея

Интуитивно предел описывает, к какому значению «стремится» функция \(f(x)\), когда аргумент \(x\) приближается к некоторой точке \(a\), но не обязательно достигает её.

NoteКлючевая идея

Нас интересует поведение функции вблизи точки, а не в самой точке. Это принципиально: функция может даже не быть определена в точке \(a\).

1.2 Формальное определение (по Коши)

Definition 1 Определение (предел по Коши). Говорят, что функция \(f(x)\) имеет предел \(L\) при \(x \to a\), если

\[\forall\,\varepsilon > 0\quad \exists\,\delta > 0\quad \forall\, x:\; 0 < |x - a| < \delta \;\Rightarrow\; |f(x) - L| < \varepsilon. \tag{1}\]

Обозначение: \(\displaystyle\lim_{x \to a} f(x) = L\).

Как читать формула 1. Для любой наперёд заданной «трубки» шириной \(2\varepsilon\) вокруг значения \(L\) можно найти «коридор» шириной \(2\delta\) вокруг точки \(a\), такой что все значения функции из этого коридора попадают в «трубку».

Caution▶ Почему условие \(0 < |x - a|\), а не просто \(|x - a|\)?

Условие \(0 < |x - a|\) означает \(x \neq a\). Это гарантирует, что мы смотрим на поведение функции вблизи точки, а не в самой точке. Значение \(f(a)\) (или его отсутствие) не влияет на предел.

Пример: функция \(f(x) = \dfrac{x^2 - 1}{x - 1}\) не определена при \(x = 1\), но \(\displaystyle\lim_{x \to 1} f(x) = 2\), поскольку при \(x \neq 1\): \[f(x) = \frac{(x-1)(x+1)}{x-1} = x + 1 \;\longrightarrow\; 2.\]

1.3 Односторонние пределы

Definition 2 Определение. - Предел справа: \(\displaystyle\lim_{x \to a^+} f(x) = L\), если \(\varepsilon\)-\(\delta\) условие выполнено при \(0 < x - a < \delta\). - Предел слева: \(\displaystyle\lim_{x \to a^-} f(x) = L\), если при \(0 < a - x < \delta\).

Theorem 1 Теорема. \(\displaystyle\lim_{x \to a} f(x) = L\) тогда и только тогда, когда \[\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = L.\]

Tip▶ Доказательство теоремы

Необходимость (\(\Rightarrow\)). Пусть \(\lim_{x\to a} f(x) = L\). По определению, для любого \(\varepsilon > 0\) найдётся \(\delta > 0\) такое, что \(0 < |x-a| < \delta \Rightarrow |f(x) - L| < \varepsilon\). В частности, это выполнено при \(0 < x - a < \delta\) и при \(0 < a - x < \delta\). Значит, оба односторонних предела равны \(L\).

Достаточность (\(\Leftarrow\)). Пусть оба предела равны \(L\). Для данного \(\varepsilon > 0\) найдём \(\delta_1, \delta_2 > 0\) из определений левого и правого пределов. Возьмём \(\delta = \min(\delta_1, \delta_2)\). Тогда при \(0 < |x - a| < \delta\) либо \(x > a\), либо \(x < a\), и в обоих случаях \(|f(x) - L| < \varepsilon\). \(\blacksquare\)

2 Теоремы о пределах

Следующие теоремы позволяют вычислять пределы, не возвращаясь к определению.

2.1 Арифметика пределов

Theorem 2 Теорема (арифметика пределов). Пусть \(\displaystyle\lim_{x \to a} f(x) = A\) и \(\displaystyle\lim_{x \to a} g(x) = B\). Тогда:

  1. \(\displaystyle\lim_{x \to a} [f(x) \pm g(x)] = A \pm B\)
  2. \(\displaystyle\lim_{x \to a} [f(x) \cdot g(x)] = A \cdot B\)
  3. \(\displaystyle\lim_{x \to a} \dfrac{f(x)}{g(x)} = \dfrac{A}{B}\), если \(B \neq 0\).
Tip▶ Доказательство (для произведения)

Нужно показать: для любого \(\varepsilon > 0\) найдётся \(\delta > 0\) такое, что \(0 < |x - a| < \delta \Rightarrow |f(x)g(x) - AB| < \varepsilon\).

Оценим: \[|f(x)g(x) - AB| = |f(x)g(x) - Ag(x) + Ag(x) - AB|\] \[\leq |g(x)|\cdot|f(x) - A| + |A|\cdot|g(x) - B|.\]

Поскольку \(g(x) \to B\), функция \(g\) ограничена вблизи \(a\): найдём \(\delta_0\) такое, что \(|g(x)| \leq |B| + 1 =: M\) при \(|x-a| < \delta_0\).

Пусть \(\varepsilon > 0\). Выберем \(\delta_1, \delta_2\) так, чтобы: \[|f(x)-A| < \frac{\varepsilon}{2M}, \quad |g(x)-B| < \frac{\varepsilon}{2(|A|+1)}.\]

Тогда при \(\delta = \min(\delta_0, \delta_1, \delta_2)\): \[|f(x)g(x) - AB| \leq M \cdot \frac{\varepsilon}{2M} + |A| \cdot \frac{\varepsilon}{2(|A|+1)} < \varepsilon. \quad \blacksquare\]

2.2 Теорема о сжатой переменной (лемма о двух милиционерах)

Theorem 3 Теорема (сжатая переменная). Если \(g(x) \leq f(x) \leq h(x)\) в некоторой проколотой окрестности точки \(a\) и \[\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L,\] то \(\displaystyle\lim_{x \to a} f(x) = L\).

Tip▶ Доказательство

Пусть \(\varepsilon > 0\). Из определений пределов \(g\) и \(h\) найдём \(\delta > 0\) такое, что при \(0 < |x - a| < \delta\): \[L - \varepsilon < g(x) \leq f(x) \leq h(x) < L + \varepsilon.\] Значит, \(|f(x) - L| < \varepsilon\). \(\blacksquare\)

3 Замечательные пределы

3.1 Первый замечательный предел

\[\lim_{x \to 0} \frac{\sin x}{x} = 1 \tag{2}\]

Tip▶ Доказательство через площади

Рассмотрим единичный круг. При \(0 < x < \pi/2\) сравним площади:

\[S_{\triangle OAB} \leq S_{\text{сектор}} \leq S_{\triangle OAT}\]

Каждая из площадей выражается через \(x\): \[\frac{\sin x}{2} \leq \frac{x}{2} \leq \frac{\tan x}{2}.\]

Делим на \(\dfrac{\sin x}{2} > 0\): \[1 \leq \frac{x}{\sin x} \leq \frac{1}{\cos x}.\]

При \(x \to 0^+\) правая часть \(\to 1\), поэтому по теореме о сжатой переменной (теорема 3) \(\dfrac{x}{\sin x} \to 1\), то есть \(\dfrac{\sin x}{x} \to 1\).

Для \(x < 0\) используем чётность функции \(\dfrac{\sin x}{x}\). \(\blacksquare\)

NoteСледствия первого замечательного предела

Из формула 2 непосредственно получаем эквивалентности при \(x \to 0\): \[\sin x \sim x, \quad \tan x \sim x, \quad \arcsin x \sim x, \quad \arctan x \sim x.\]

3.2 Второй замечательный предел

\[\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e \approx 2{,}71828\ldots \tag{3}\]

Tip▶ Существование предела (набросок доказательства)

Обозначим \(a_n = \left(1 + \dfrac{1}{n}\right)^n\) для натуральных \(n\).

Монотонность. По неравенству AM–GM: \[a_n = \left(1 + \frac{1}{n}\right)^n < \left(1 + \frac{1}{n}\right)^{n+1} = a_{n+1} \cdot \frac{n}{n+1}.\] (Более аккуратное доказательство через логарифмическую выпуклость.)

Ограниченность. По формуле бинома Ньютона: \[a_n = \sum_{k=0}^{n} \binom{n}{k} \frac{1}{n^k} \leq \sum_{k=0}^{n} \frac{1}{k!} < e^1 = e < 3.\]

По теореме Вейерштрасса монотонная ограниченная последовательность сходится. Её предел и называется числом \(e\). \(\blacksquare\)

4 Непрерывность функции

4.1 Определение

Definition 3 Определение. Функция \(f\) непрерывна в точке \(a\), если:

  1. \(f\) определена в точке \(a\),
  2. существует \(\displaystyle\lim_{x \to a} f(x)\),
  3. \(\displaystyle\lim_{x \to a} f(x) = f(a)\).

Все три условия можно записать одним равенством: \[\lim_{x \to a} f(x) = f(a), \tag{4}\] или в терминах приращений (\(\Delta x = x - a\), \(\Delta y = f(a + \Delta x) - f(a)\)): \[\lim_{\Delta x \to 0} \Delta y = 0.\]

4.2 Классификация точек разрыва

Если \(f\) не непрерывна в точке \(a\), то \(a\) — точка разрыва.

Note▶ Подробнее про устранимый разрыв

Если \(\lim_{x \to a} f(x) = L\), но \(f(a) \neq L\) (или \(f\) не определена в \(a\)), то разрыв устраняется: достаточно доопределить \(f(a) := L\).

Пример: \(f(x) = \dfrac{\sin x}{x}\) при \(x \neq 0\). По формула 2 предел равен \(1\), но в точке \(x = 0\) функция не определена. Полагая \(f(0) = 1\), получаем непрерывную функцию.

4.3 Теорема о промежуточном значении (Больцано–Коши)

Theorem 4 Теорема. Если \(f\) непрерывна на \([a, b]\) и \(f(a) \cdot f(b) < 0\) (значения на концах разных знаков), то существует \(c \in (a, b)\) такое, что \(f(c) = 0\).

Tip▶ Доказательство методом деления пополам

Построим последовательность вложенных отрезков. Пусть \(a_0 = a\), \(b_0 = b\). На \(k\)-м шаге возьмём середину \(m_k = \dfrac{a_k + b_k}{2}\):

  • Если \(f(m_k) = 0\) — нашли \(c = m_k\), готово.
  • Если \(f(a_k) \cdot f(m_k) < 0\) — полагаем \(a_{k+1} = a_k\), \(b_{k+1} = m_k\).
  • Иначе — \(a_{k+1} = m_k\), \(b_{k+1} = b_k\).

Длина отрезка \([a_k, b_k]\) равна \(\dfrac{b-a}{2^k} \to 0\). По лемме о вложенных отрезках существует единственная точка \(c \in \bigcap_k [a_k, b_k]\).

По построению \(f(a_k) \cdot f(b_k) < 0\) для всех \(k\). Так как \(a_k \to c\) и \(b_k \to c\), а \(f\) непрерывна, то \(f(a_k) \to f(c)\) и \(f(b_k) \to f(c)\). Из \(f(a_k) \cdot f(b_k) < 0\) при переходе к пределу получаем \(f(c)^2 \leq 0\), значит \(f(c) = 0\). \(\blacksquare\)

5 Задачи для самопроверки

WarningЗадачи

Вычислите пределы:

  1. \(\displaystyle\lim_{x \to 2} \frac{x^2 - 4}{x - 2}\)

  2. \(\displaystyle\lim_{x \to 0} \frac{\sin 3x}{5x}\)

  3. \(\displaystyle\lim_{x \to \infty} \left(1 + \frac{2}{x}\right)^{3x}\)

  4. \(\displaystyle\lim_{x \to 0} \frac{1 - \cos x}{x^2}\)

Классифицируйте точки разрыва:

  1. \(f(x) = \dfrac{x^2 - 1}{|x - 1|}\)

  2. \(g(x) = \begin{cases} x \sin\dfrac{1}{x}, & x \neq 0 \\ 1, & x = 0 \end{cases}\)

Note▶ Ответы
  1. \(4\)
  2. \(\dfrac{3}{5}\) (по первому замечательному пределу)
  3. \(e^6\) (по второму замечательному пределу: \(\left(1+\frac{2}{x}\right)^{3x} = \left[\left(1+\frac{2}{x}\right)^{x/2}\right]^6 \to e^6\))
  4. \(\dfrac{1}{2}\) (классический результат, доказывается через первый замечательный предел или эквивалентности)
  5. Точка разрыва \(x = 1\): слева \(f \to -2\), справа \(f \to 2\) — разрыв I рода (скачок)
  6. Точка \(x = 0\) — устранимый разрыв: \(\lim_{x\to 0} x\sin\tfrac{1}{x} = 0 \neq g(0) = 1\)

# Предел функции {#sec-limit-def}

## Неформальная идея

Интуитивно предел описывает, к какому значению «стремится» функция $f(x)$,
когда аргумент $x$ приближается к некоторой точке $a$, но **не обязательно
достигает её**.

::: {.callout-note}
## Ключевая идея
Нас интересует поведение функции **вблизи** точки, а не **в самой** точке.
Это принципиально: функция может даже не быть определена в точке $a$.
:::

## Формальное определение (по Коши) {#sec-cauchy-def}

::: {#def-limit}
**Определение (предел по Коши).**
Говорят, что функция $f(x)$ имеет предел $L$ при $x \to a$, если

$$\forall\,\varepsilon > 0\quad \exists\,\delta > 0\quad \forall\, x:\;
  0 < |x - a| < \delta \;\Rightarrow\; |f(x) - L| < \varepsilon.$$  {#eq-limit-cauchy}

Обозначение: $\displaystyle\lim_{x \to a} f(x) = L$.
:::

**Как читать @eq-limit-cauchy.**
Для любой наперёд заданной «трубки» шириной $2\varepsilon$ вокруг значения $L$
можно найти «коридор» шириной $2\delta$ вокруг точки $a$, такой что все значения
функции из этого коридора попадают в «трубку».

::: {.callout-caution collapse="true"}
## ▶ Почему условие $0 < |x - a|$, а не просто $|x - a|$?

Условие $0 < |x - a|$ означает $x \neq a$.
Это гарантирует, что мы смотрим на поведение функции **вблизи** точки, а
не в самой точке. Значение $f(a)$ (или его отсутствие) не влияет на предел.

Пример: функция $f(x) = \dfrac{x^2 - 1}{x - 1}$ не определена при $x = 1$,
но $\displaystyle\lim_{x \to 1} f(x) = 2$, поскольку при $x \neq 1$:
$$f(x) = \frac{(x-1)(x+1)}{x-1} = x + 1 \;\longrightarrow\; 2.$$
:::

## Односторонние пределы

::: {#def-one-sided}
**Определение.**
- *Предел справа*: $\displaystyle\lim_{x \to a^+} f(x) = L$, если $\varepsilon$-$\delta$
  условие выполнено при $0 < x - a < \delta$.
- *Предел слева*: $\displaystyle\lim_{x \to a^-} f(x) = L$, если при $0 < a - x < \delta$.
:::

::: {#thm-two-sided}
**Теорема.**
$\displaystyle\lim_{x \to a} f(x) = L$ тогда и только тогда, когда
$$\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = L.$$
:::

::: {.callout-tip collapse="true"}
## ▶ Доказательство теоремы

**Необходимость** ($\Rightarrow$). Пусть $\lim_{x\to a} f(x) = L$.
По определению, для любого $\varepsilon > 0$ найдётся $\delta > 0$ такое, что
$0 < |x-a| < \delta \Rightarrow |f(x) - L| < \varepsilon$.
В частности, это выполнено при $0 < x - a < \delta$ и при $0 < a - x < \delta$.
Значит, оба односторонних предела равны $L$.

**Достаточность** ($\Leftarrow$). Пусть оба предела равны $L$.
Для данного $\varepsilon > 0$ найдём $\delta_1, \delta_2 > 0$ из
определений левого и правого пределов. Возьмём $\delta = \min(\delta_1, \delta_2)$.
Тогда при $0 < |x - a| < \delta$ либо $x > a$, либо $x < a$, и в обоих случаях
$|f(x) - L| < \varepsilon$. $\blacksquare$
:::

# Теоремы о пределах {#sec-limit-theorems}

Следующие теоремы позволяют вычислять пределы, не возвращаясь к определению.

## Арифметика пределов

::: {#thm-arithmetic}
**Теорема (арифметика пределов).**
Пусть $\displaystyle\lim_{x \to a} f(x) = A$ и $\displaystyle\lim_{x \to a} g(x) = B$. Тогда:

1. $\displaystyle\lim_{x \to a} [f(x) \pm g(x)] = A \pm B$
2. $\displaystyle\lim_{x \to a} [f(x) \cdot g(x)] = A \cdot B$
3. $\displaystyle\lim_{x \to a} \dfrac{f(x)}{g(x)} = \dfrac{A}{B}$, если $B \neq 0$.
:::

::: {.callout-tip collapse="true"}
## ▶ Доказательство (для произведения)

Нужно показать: для любого $\varepsilon > 0$ найдётся $\delta > 0$ такое, что
$0 < |x - a| < \delta \Rightarrow |f(x)g(x) - AB| < \varepsilon$.

Оценим:
$$|f(x)g(x) - AB| = |f(x)g(x) - Ag(x) + Ag(x) - AB|$$
$$\leq |g(x)|\cdot|f(x) - A| + |A|\cdot|g(x) - B|.$$

Поскольку $g(x) \to B$, функция $g$ ограничена вблизи $a$:
найдём $\delta_0$ такое, что $|g(x)| \leq |B| + 1 =: M$ при $|x-a| < \delta_0$.

Пусть $\varepsilon > 0$. Выберем $\delta_1, \delta_2$ так, чтобы:
$$|f(x)-A| < \frac{\varepsilon}{2M}, \quad |g(x)-B| < \frac{\varepsilon}{2(|A|+1)}.$$

Тогда при $\delta = \min(\delta_0, \delta_1, \delta_2)$:
$$|f(x)g(x) - AB| \leq M \cdot \frac{\varepsilon}{2M} + |A| \cdot \frac{\varepsilon}{2(|A|+1)} < \varepsilon. \quad \blacksquare$$
:::

## Теорема о сжатой переменной (лемма о двух милиционерах)

::: {#thm-squeeze}
**Теорема (сжатая переменная).**
Если $g(x) \leq f(x) \leq h(x)$ в некоторой проколотой окрестности точки $a$ и
$$\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L,$$
то $\displaystyle\lim_{x \to a} f(x) = L$.
:::

::: {.callout-tip collapse="true"}
## ▶ Доказательство

Пусть $\varepsilon > 0$. Из определений пределов $g$ и $h$ найдём $\delta > 0$
такое, что при $0 < |x - a| < \delta$:
$$L - \varepsilon < g(x) \leq f(x) \leq h(x) < L + \varepsilon.$$
Значит, $|f(x) - L| < \varepsilon$. $\blacksquare$
:::

# Замечательные пределы {#sec-remarkable}

## Первый замечательный предел

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$ {#eq-first-remarkable}

::: {.callout-tip collapse="true"}
## ▶ Доказательство через площади

Рассмотрим единичный круг. При $0 < x < \pi/2$ сравним площади:

$$S_{\triangle OAB} \leq S_{\text{сектор}} \leq S_{\triangle OAT}$$

Каждая из площадей выражается через $x$:
$$\frac{\sin x}{2} \leq \frac{x}{2} \leq \frac{\tan x}{2}.$$

Делим на $\dfrac{\sin x}{2} > 0$:
$$1 \leq \frac{x}{\sin x} \leq \frac{1}{\cos x}.$$

При $x \to 0^+$ правая часть $\to 1$, поэтому по теореме о сжатой переменной
(@thm-squeeze) $\dfrac{x}{\sin x} \to 1$, то есть $\dfrac{\sin x}{x} \to 1$.

Для $x < 0$ используем чётность функции $\dfrac{\sin x}{x}$. $\blacksquare$
:::

::: {.callout-note}
## Следствия первого замечательного предела
Из @eq-first-remarkable непосредственно получаем эквивалентности при $x \to 0$:
$$\sin x \sim x, \quad \tan x \sim x, \quad \arcsin x \sim x, \quad \arctan x \sim x.$$
:::

## Второй замечательный предел

$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e \approx 2{,}71828\ldots$$ {#eq-second-remarkable}

::: {.callout-tip collapse="true"}
## ▶ Существование предела (набросок доказательства)

Обозначим $a_n = \left(1 + \dfrac{1}{n}\right)^n$ для натуральных $n$.

**Монотонность.** По неравенству AM–GM:
$$a_n = \left(1 + \frac{1}{n}\right)^n < \left(1 + \frac{1}{n}\right)^{n+1} = a_{n+1} \cdot \frac{n}{n+1}.$$
(Более аккуратное доказательство через логарифмическую выпуклость.)

**Ограниченность.** По формуле бинома Ньютона:
$$a_n = \sum_{k=0}^{n} \binom{n}{k} \frac{1}{n^k}
      \leq \sum_{k=0}^{n} \frac{1}{k!} < e^1 = e < 3.$$

По теореме Вейерштрасса монотонная ограниченная последовательность сходится.
Её предел и называется числом $e$. $\blacksquare$
:::

# Непрерывность функции {#sec-continuity}

## Определение

::: {#def-continuity}
**Определение.**
Функция $f$ **непрерывна в точке** $a$, если:

1. $f$ определена в точке $a$,
2. существует $\displaystyle\lim_{x \to a} f(x)$,
3. $\displaystyle\lim_{x \to a} f(x) = f(a)$.

Все три условия можно записать одним равенством:
$$\lim_{x \to a} f(x) = f(a),$$ {#eq-continuity}
или в терминах приращений ($\Delta x = x - a$, $\Delta y = f(a + \Delta x) - f(a)$):
$$\lim_{\Delta x \to 0} \Delta y = 0.$$
:::

## Классификация точек разрыва {#sec-discontinuity}

Если $f$ **не** непрерывна в точке $a$, то $a$ — **точка разрыва**.

| Тип разрыва | Условие | Пример |
|-------------|---------|--------|
| Устранимый | $\lim_{a^-} = \lim_{a^+} \neq f(a)$ | $\frac{\sin x}{x}$ в точке $0$ |
| I рода (скачок) | $\lim_{a^-} \neq \lim_{a^+}$, оба конечны | $\text{sgn}(x)$ в точке $0$ |
| II рода | хотя бы один из пределов $= \pm\infty$ или не существует | $\frac{1}{x}$ в точке $0$ |

: Классификация точек разрыва {#tbl-discontinuity}

::: {.callout-note collapse="true"}
## ▶ Подробнее про устранимый разрыв

Если $\lim_{x \to a} f(x) = L$, но $f(a) \neq L$ (или $f$ не определена в $a$),
то разрыв **устраняется**: достаточно доопределить $f(a) := L$.

Пример: $f(x) = \dfrac{\sin x}{x}$ при $x \neq 0$. 
По @eq-first-remarkable предел равен $1$, но в точке $x = 0$ функция не определена.
Полагая $f(0) = 1$, получаем непрерывную функцию.
:::

## Теорема о промежуточном значении (Больцано–Коши)

::: {#thm-ivt}
**Теорема.**
Если $f$ непрерывна на $[a, b]$ и $f(a) \cdot f(b) < 0$
(значения на концах разных знаков), то существует $c \in (a, b)$ такое, что $f(c) = 0$.
:::

::: {.callout-tip collapse="true"}
## ▶ Доказательство методом деления пополам

Построим последовательность вложенных отрезков.
Пусть $a_0 = a$, $b_0 = b$. На $k$-м шаге возьмём середину $m_k = \dfrac{a_k + b_k}{2}$:

- Если $f(m_k) = 0$ — нашли $c = m_k$, готово.
- Если $f(a_k) \cdot f(m_k) < 0$ — полагаем $a_{k+1} = a_k$, $b_{k+1} = m_k$.
- Иначе — $a_{k+1} = m_k$, $b_{k+1} = b_k$.

Длина отрезка $[a_k, b_k]$ равна $\dfrac{b-a}{2^k} \to 0$.
По лемме о вложенных отрезках существует единственная точка
$c \in \bigcap_k [a_k, b_k]$.

По построению $f(a_k) \cdot f(b_k) < 0$ для всех $k$.
Так как $a_k \to c$ и $b_k \to c$, а $f$ непрерывна, то
$f(a_k) \to f(c)$ и $f(b_k) \to f(c)$.
Из $f(a_k) \cdot f(b_k) < 0$ при переходе к пределу получаем $f(c)^2 \leq 0$,
значит $f(c) = 0$. $\blacksquare$
:::

# Задачи для самопроверки {#sec-exercises}

::: {.callout-warning}
## Задачи

**Вычислите пределы:**

(@) $\displaystyle\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$

(@) $\displaystyle\lim_{x \to 0} \frac{\sin 3x}{5x}$

(@) $\displaystyle\lim_{x \to \infty} \left(1 + \frac{2}{x}\right)^{3x}$

(@) $\displaystyle\lim_{x \to 0} \frac{1 - \cos x}{x^2}$

**Классифицируйте точки разрыва:**

(@) $f(x) = \dfrac{x^2 - 1}{|x - 1|}$

(@) $g(x) = \begin{cases} x \sin\dfrac{1}{x}, & x \neq 0 \\ 1, & x = 0 \end{cases}$
:::

::: {.callout-note collapse="true"}
## ▶ Ответы

1. $4$
2. $\dfrac{3}{5}$ (по первому замечательному пределу)
3. $e^6$ (по второму замечательному пределу: $\left(1+\frac{2}{x}\right)^{3x} = \left[\left(1+\frac{2}{x}\right)^{x/2}\right]^6 \to e^6$)
4. $\dfrac{1}{2}$ (классический результат, доказывается через первый замечательный предел или эквивалентности)
5. Точка разрыва $x = 1$: слева $f \to -2$, справа $f \to 2$ — разрыв I рода (скачок)
6. Точка $x = 0$ — устранимый разрыв: $\lim_{x\to 0} x\sin\tfrac{1}{x} = 0 \neq g(0) = 1$
:::
